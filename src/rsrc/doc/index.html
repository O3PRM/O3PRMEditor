User Manual

Introduction

O3PRM Editor is an open source software under the GPL V3 licence.

It has been developped by Vincent Renaudineau, Lionel Torti and Pierre-Henri Wuillemin for the SKOOP ANR project.

It is an IDE, Integrated Development Environment, for the O3PRM language.

Basic Usage

Click on the “New” icon to start a new project.

Use the left panel to add files by right clicking on nodes in the treeview.

For inference, simply right click on a request file (with the .o3prmr extension) and click on “execute”.



For more information you can visit aGrUM website: http://agrum.lip6.fr or the O3PRM website: http://o3prm.lip6.fr .
